# Net-Empregos Job Scraper
This is an automated Python tool using Selenium to find job vacancies in Portugal.

## Motivation
This project automates the tedious task of searching for work by scraping titles and links from net-empregos.com automatically.

## Tech Stack
Python 3, Selenium WebDriver, Webdriver-Manager, and Dotenv.

## Setup Instructions
First clone the repository to your local machine. Create a virtual environment and activate it. Install the requirements using pip install -r requirements.txt. Ensure you have Google Chrome installed.

## Usage
Run the script using the command: python main.py. The script will open the browser and begin the extraction process.

## Security
The included gitignore file protects your environment variables and local folders from being published.
